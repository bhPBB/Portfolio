Extrair Texto de PDF / Imagem

Um site onde você pode extrair textos de PDFs e imagens usando a API do Google Cloud Vision. Após o upload, a imagem é processada pela API e o texto detectado é exibido ao usuário. No caso de um PDF, após o upload, eles é, primeiramente, armazenado em um bucket do Cloud Storage. Após isso, é processados pela API, o texto detectado é exibido ao usuário e o PDF é deletado do bucket. O backend do site foi desenvolvido com Django e o frontend com HTML, CSS e Bootstrap.

IFSP Blazor

O site do IFSP - câmpus Hortolândia reimaginado. Ele contém várias informações e seções do site original do IFSP, como uma lista dos cursos técnicos e de gradução oferecidos, informações sobre a coordenadoria de extensão, uma seção onde é exibida a localização do câmpus no mapa por meio da API do google maps, e muito mais. Além de HTML e CSS, os componentes da interface foram desenvolvidos com C# Blazor e Bootstrap. O projeto está hospedado no GitHub Pages.
